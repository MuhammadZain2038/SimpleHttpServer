Simple HTTP Web Server

Welcome to the Simple HTTP Web Server!

This server is designed to provide a lightweight, easy-to-use platform for handling basic web operations.

Features

HTTP Protocol Support

  • Fully supports handling of GET and POST HTTP methods.

User Authentication System

  • Login: Allows registered users to securely log into the system. Validates user credentials against stored data.
  • Registration: Enables new users to create accounts. Stores user information securely for future logins.

Functionality Overview

Handling GET Requests

  • Serves static HTML files (e.g., home page, login page, registration form).
  • Provides easy navigation for users accessing the server through a web browser.

Handling POST Requests

  • User Login: Authenticates submitted username and password.
  • User Registration: Saves new user details securely.

File Structure

/server/
|
|-- index.html          
|-- login.html          
|-- register.html       
|-- server.cpp          
|-- users.txt           
`-- static/

Security Notes

  • Basic password storage (enhanced with SHA-256 hashing).
  • Basic input validation to prevent injection attacks (recommended to extend).

How to Use

Start the server:
g++ server.cpp -o server -pthread -lssl -lcrypto
./server

Open a browser and navigate to:
http://localhost:9090/

Use the interface to either:
- Register a new user
- Login with existing credentials
- Browse served pages

Note: This server is intended for learning and prototyping purposes. For production-level deployment, additional security and scalability features are recommended.
